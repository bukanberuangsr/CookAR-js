# CookAR-js

> A desparate take on mobile AR app

## Deskripsi

CookAR merupakan sebuah aplikasi menampilkan informasi detail seperti bahan utama makanan pada sebuah menu menggunakan marker khusus.

## Kontribusi

Proyek ini menggunakan aframe versi terbaru. Dikarenakan proyek ini memiliki model 3D, jalankan proyek ini menggunakan web server seperti apache (xampp), nginx, atau menggunakan extension live server di VSCode.

```sh
git clone https://github.com/bukanberuangsr/CookAR-js.git
cd CookAR-js
code .
```
